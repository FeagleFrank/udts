

# ORACLE源

UDTS支持Oracle数据库作为数据传输源

## ORACLE源填写表单

| 参数名   | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| IP       | 提供内网地址和外网地址两种方式传输，内网地址需要填写VPC和子网信息。 |
| 数据库名 | ORACLE数据库名称                                              |
| 端口     | ORACLE连接端口                                                |
| 用户名   | ORACLE连接用户名                                              |
| 密码     | ORACLE数据库对应用户密码                                      |
| 表名     | ORACLE传输表名，若不填，系统默认为整库迁移                    |
