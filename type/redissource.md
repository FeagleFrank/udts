

# Redis源 

UDTS可以将Redis作为数据源进行全量+增量任务的传输，目前支持集群版和主备版(暂不支持UCloud Redis分布式版本)。

## Redis源填写表单

| 参数名           | 说明                                                         |
| ---------------- | ------------------------------------------------------------ |
|地址     | 数据库地址，只填写主(master)地址，集群模式下，多个地址用 ; 相连 |
|密码 |Redis密码，非必填  |
| Redis类型             | 集群版/主备版                                            |
| VPC ID       | VPC ID |
| 所属子网         | VPC下所属子网 |
