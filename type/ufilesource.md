

# UFile源

UDTS支持UFile-bucket作为数据传输源

## UFile源填写表单

| 参数名   | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| Bucket名称       | Bucket 名称 |
| 用户自定义域名 | UFile bucket 用户定义的域名|                                         |
| 用户私钥     | 为获得用户上传下载接口执行权限，需要用户提供key，可在UAPI控制台获取。                                                |
| 用户公钥   | 为获得用户上传下载接口执行权限，需要用户提供key，可在UAPI控制台获取。                                              |
| Prefix     |    增量任务使用，用于用户区分增量的分区。                                   |
